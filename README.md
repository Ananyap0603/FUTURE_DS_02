# Customer Retention & Churn Analysis

## Overview

This project analyzes churn patterns and retention trends for a
subscription-based SaaS company. The objective is to identify key churn
drivers, retention cohorts, and customer lifetime insights.

## Project Goals

-   Analyze churn rates and retention trends
-   Perform cohort analysis by signup month, plan type, and region
-   Estimate Customer Lifetime Value (LTV)
-   Identify key retention drivers
-   Provide actionable SaaS business recommendations

## Key Findings

-   Overall churn rate: 26.4%
-   Monthly churn rate: 4.1%
-   Annual plan churn significantly lower than monthly
-   Early engagement strongly predicts long-term retention
-   India region shows higher price-sensitive churn

## Tools Used

-   Python (Data Cleaning & Analysis)
-   Power BI / Tableau (Dashboard Design Concept)
-   Excel (Initial Data Structuring)

## Business Recommendations

1.  Improve onboarding and first 30-day engagement.
2.  Incentivize annual plan subscriptions.
3.  Implement churn prediction modeling.
4.  Introduce activation KPI tracking.
5.  Apply regional pricing optimization.

## Deliverables

-   PDF Business Report
-   Dashboard Structure Plan
-   Strategic SaaS Advisory Insights
